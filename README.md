# Analyse de la biodiversité urbaine – Moineaux domestiques

## Objectif
Ce projet analyse les observations de moineaux domestiques dans le territoire de Grand Paris Seine Ouest à partir de données ouvertes (période de mars à mai).

## Données
Source : data.gouv.fr et https://data.seineouest.fr/explore/dataset/moineaux-domestiques-mdna/custom/?disjunctive.nb_individus&disjunctive.commune

## Méthodologie
- Nettoyage des données
- Analyse exploratoire
- Analyse saisonnière
- Analyse par commune
- Visualisations (courbes, heatmap)
- Prévision simple (régression linéaire, moyenne mobile)

## Résultats
- Variation saisonnière observée
- Différences entre communes
- Tendance globale à la baisse (à interpréter avec prudence)

## Limites
- Données sur quelques mois
- Biais d’observation possibles

## Perspectives
- Ajouter des études longitudinales
- Intégrer des données climatiques et environnementales
- Comparer avec d’autres territoires
- Comparer le nombre d'habitants par commune par le nombre d'observateurs uniques
