# Analyse de la biodiversité urbaine – Moineaux domestiques

## Objectif
Ce projet analyse les observations de moineaux domestiques dans le territoire de Grand Paris Seine Ouest à partir de données ouvertes (période de 1 an).

## Données
Source : data.gouv.fr

## Méthodologie
- Nettoyage des données
- Analyse exploratoire
- Analyse saisonnière
- Analyse par commune
- Visualisations (courbes, heatmap)
- Prévision simple (régression linéaire, moyenne mobile)

## Résultats
- Variation saisonnière observée
- Différences entre communes
- Tendance globale à la baisse (à interpréter avec prudence)

## Limites
- Données sur une seule année
- Biais d’observation possibles

## Perspectives
- Ajouter plusieurs années
- Intégrer météo et environnement
- Comparaison avec d’autres territoires
