# Analyse de la biodiversité urbaine – Moineaux domestiques

## Objectif
Ce projet analyse les observations de moineaux domestiques dans le territoire de Grand Paris Seine Ouest à partir de données ouvertes (période de mars à mai).

L'intérêt d'une telle analyse est d'aider les territoires à prendre des décisions concernant la végétalisation dans un milieu urbain.

## Données
Source : data.gouv.fr et https://data.seineouest.fr/explore/dataset/moineaux-domestiques-mdna/custom/?disjunctive.nb_individus&disjunctive.commune

## Méthodologie
- Nettoyage des données
- Analyse exploratoire
- Analyse saisonnière
- Analyse par commune
- Visualisations (courbes, heatmap)
- Prévision simple (régression linéaire, moyenne mobile)

## Résultats
- Variation saisonnière observée
- Différences entre communes
- Tendance globale à la baisse (à interpréter avec prudence)

## Limites
- Données sur quelques mois
- Biais d’observation possibles

## Perspectives
- Ajouter des études longitudinales
- Intégrer des données climatiques et environnementales
- Comparer avec d’autres territoires
- Comparer le nombre d'habitants par commune par le nombre d'observateurs uniques

# Prototype d' application - URBOC 

## Concept
Une application d’observation de biodiversité sans inscription, basée sur la géolocalisation, qui permet de contribuer instantanément à un score d'observation locale des espèces.
Chaque observation génère une "récompense" immédiate avec un message positif.
Des notifications incitent les observateurs à poursuivre leur contribution en science participative.

## Valeur ajoutée
Pour les observateurs : observer la biodiversité en ville, sans effort, et voir immédiatement l’impact de ses observations sur son territoire, sans créer de compte.
L'objectif est de donner à n'importe quel public l'envie de participer durablement.

Pour les territoires : la possibilité de mutualiser les opérations d'observation sur une seule application et visualiser les données des autres territoires en temps réel.
